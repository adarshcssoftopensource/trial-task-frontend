# React + Vite

## Prerequisite 
- Node.JS v18.x 
- React 18

## Install package
> yarn

## Run reactjs
> yarn run dev

## Build
> yarn run build

## Local start
- run `git pull` (pull latest changes)
- add `VITE_KEY_BASE_URL=http://localhost:5000/api` in .env file
- run `yarn` (install all packages)
- run `yarn run dev` (start local server) (see "Starting the local server")
